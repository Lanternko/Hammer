# 🔨 Roguelike Numerical Showdown - 重錘征服者

## 🎮 遊戲概述

一款基於純數值計算的網頁單機 Roguelike 遊戲，以**重錘BD系統**為核心玩法。玩家操控重錘戰士，通過20個關卡的征程，收集徽章構建流派，最終成為重錘之王！

### 🌟 核心特色

- **🔨 重錘技能樹**: 7種重錘徽章組成完整的技能分支系統
- **⚖️ 智慧平衡**: 三參數系統確保每一關都有恰當的挑戰性
- **🎯 戰力統一**: 開根號顯示模式，直觀展示實力對比
- **🎖️ 多元流派**: 反甲流、爆發流、控制流等不同BD路線
- **🤖 AI驗證**: 自動化測試確保遊戲平衡性

## 🎯 遊戲玩法

### 戰鬥機制
- **20 FPS戰鬥系統**: 精確的幀級戰鬥計算
- **傷害公式**: `實際傷害 = max(0, (原始傷害 / (1 + 護甲/100)) - 固傷減免)`
- **攻擊節奏**: 每個角色都有獨特的攻擊速度和攻擊幀數
- **暴擊系統**: 基礎10%暴擊率，可通過徽章提升

### 重錘BD系統 🔨

#### 基礎技能
- **🔨 重錘精通**: 每次攻擊15%機率造成200%傷害（開局贈送）

#### 攻擊分支
- **🌪️ 重錘風暴**: 重錘觸發時，下次攻擊必定暴擊
- **🔥 重錘狂怒**: 重錘觸發時，攻速+50%持續3秒

#### 防禦分支  
- **🛡️ 重錘護盾**: 重錘觸發時，獲得30護甲持續5秒
- **💚 重錘恢復**: 重錘觸發時，回復15點生命值

#### 強化分支
- **⚡ 重錘加重**: 重錘觸發率提升至25%，傷害提升至250%
- **⏱️ 重錘延續**: 重錘眩暈時間延長至2秒（眩暈功能可關閉）

### 多元流派構建

#### 🛡️ 反甲流（固傷堆疊）
```
核心徽章: 重錘護盾 + 反甲護盾 + 護甲強化
特色: 每受傷5次反彈敵人5%血量，無視護甲
適合: 穩健推進，適合新手
```

#### ⚡ 爆發流（秒殺流）
```
核心徽章: 重錘風暴 + 重錘狂怒 + 暴擊強化
特色: 高頻暴擊配合攻速爆發
適合: 激進玩法，高風險高回報
```

#### 🎯 控制流（眩暈鎖定）
```
核心徽章: 重錘延續 + 重錘護盾 + 吸血
特色: 眩暈控制+生存能力並重
適合: 平衡型玩法，容錯率高
```

## 📊 遊戲數據

### 關卡設計
- **總關卡數**: 20關
- **Boss關**: 第5、10、15、20關（里程碑徽章獎勵）
- **事件關**: 第3、8、13、18關（商店購買）
- **戰力成長**: 每關敵人戰力提升約10%

### 經濟系統
- **金幣獲得**: 每關1金，Boss關2金，利息最多3金/回合
- **徽章獲得**: Boss關里程碑獎勵 + 升級選擇 + 商店購買
- **鑽石系統**: 每5關獲得1鑽石（用於未來擴展）

### 戰力計算公式
```javascript
// 統一戰力顯示（開根號模式）
DPS = 攻擊力 × 攻擊速度 × (1 + 暴擊率 × 暴擊倍率)
EHP = 生命值 / (1 - 護甲減傷率) × 其他生存修正
戰力 = √(DPS × EHP)
顯示戰力 = Math.floor(戰力)
```
